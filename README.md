# Truckload

Moving your video collection with ease

<img src="public/truckload.png" alt="Truckload" width="200px">

## How to use

### Start development server

`yarn dev`

### Run local Inngest server

`npx inngest-cli@latest dev`

### Run partykit server

`cd app && npx partykit dev`
